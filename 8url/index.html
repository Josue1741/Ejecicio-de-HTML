Las URLs

Las urls son las direcciones a las páginas website en donde podemos consultar los archivos de dichas páginas.
Sus siglas: Uniform Resource Locator(localizador de recursos uniformes).
Para las máquinas la dirección real es la IP solo que dichas IP´s se traducen en nombres DNS por ejemplo www.google.com. Que es mucho más sencillo para los humanos que recordar que 173.194.76.138.

Las partes de las que se compone una URL son:

https://www.cocacola.es/es/productos-coca-cola/

ESQUEMA_PROTOCOLO_HTTP://NOMBRE_DEL_HOST/RUTA/NOMBRE_DEL_ARCHIVO

cocacola.escocacola.es
